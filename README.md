# adamturaj.com

This is the rewrite of my portfolio utilizing react-three-fiber and Next.JS

## Getting started

Install the dependencies:
```bash
yarn install
```

Run the server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then have fun!
